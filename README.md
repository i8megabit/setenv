# setenv

Набор скриптов для работы с локальным окружением. Здесь есть docker-compose и команды-помощники.

## Что умеет

- `setenv` добавляет переменные и делает команды (`docker-stack`, `ansible-run` и другие) видимыми в PATH.
- В `docker-compose` описаны контейнеры Traefik и Ansible.
- В `.helpers` лежат скрипты для запуска и перезапуска контейнера Ansible.

## Как пользоваться

1. Запусти `./setenv`. В домашней папке появится `~/bin` и новые команды.
2. Чтобы поднять сервисы:
   ```bash
   docker-stack up -d
   ```
3. Чтобы зайти в контейнер Ansible:
   ```bash
   ansible-run
   ```
4. Чтобы сменить роль и перезапустить контейнер:
   ```bash
   ansible-role <имя роли>
   ansible-restart <имя роли>
   ```

Файл `.env` хранит переменные. Настрой их под себя.

Текущая версия лежит в файле `VERSION`.
